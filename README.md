# Money Manager

## RWAapp


## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve     localhost/8080/public
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start  
```
### JSON-server localhost:3000/db
```sh
$ json-server db.json
```

### Build
Compile and build

```sh
$ npm run build
```
